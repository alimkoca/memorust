# memorust
Simple and stupid flash card implementation on CLI

How to add a card:
```
$ # Running with binary file
$ memorust add "Who is father of Turkey?" "Atatürk"

$ # Running with cargo
$ cargo -- add "Who is father of Turkey?" "Atatürk"
```

Test your knowledge:
```
$ # Running with binary file
$ memorust test

$ # Running with cargo
$ cargo -- test
```
